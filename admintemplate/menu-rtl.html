CSS Class
To use static layout for your project add .menu-rtl class in body tag.
HTML Markup

Use the following code to use RTL Menu

Usage:
<!DOCTYPE html>
<html lang="en">
    <head></head>
    <body class="menu-rtl">
    <div class="card">
      <div class="card-header">
        <h5>Hello card</h5>

        <div class="card-header-right">
            <i class="icofont icofont-rounded-down"></i>
            <i class="icofont icofont-refresh"></i>
            <i class="icofont icofont-close-circled"></i>
        </div>
      </div>
